# 🥗 NutriCart

An AI-powered health & nutrition e-commerce platform built using the MERN stack.

## 🚀 Features

- 🔐 OTP-based authentication
- 🛒 Vegan & protein product marketplace
- 🤖 AI-based diet recommendations
- 🏋️ Personalized exercise suggestions
- 📊 Admin dashboard

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- AI: Custom recommendation system

## 📂 Project Structure

NutriCart/
├── frontend/
├── backend/
├── README.md

## ⚙️ Setup Instructions

### 1. Clone the repo

git clone https://github.com/suryansh-devlab/NutriCart.git

### 2. Install dependencies

cd backend
npm install

cd ../frontend
npm install

### 3. Run the app

npm run dev

## 🔐 Environment Variables

Create a `.env` file in backend:
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
